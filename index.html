User Documentation

This is experimental - you can trade tokens against SOV without explicit trading fees. While there are no trading fees, there are still token burning fees (up to 1 percent) when transferring SOV tokens.

Over time, these fees will decrease as SOV supply decreases. From a supply of 21 million, the transfer of SOV tokens will take place without burning. From this point on, sovorderbook will be free of charge. It is only planned to unlock some additional functions against SOV burning.

The contract is limited to 4 central functions for creating and taking trading orders. This frontend will use the information about the orders and create a list of bundled transactions.

sovorderbook (beta) is an experimental trading platform. The smartcontracts have not been audited by an external team. While there are no trading fees, there are still token burning fees (up to 1%) when transferring SOV tokens.

This is a community driven project and not part of the official SOV team.

The frontend is designed to be operated locally on your own computer (no server-side database). Please make sure to keep the software up to date by regularly updating the code from the official Github repository.

Further information and details can be found in the menu under "user documentation".
